## New page structure (Stripe/Linear/Vercel: centered, spacious, single-column-leaning)

1. **Background layer** — `#08090A` base. One radial gradient mesh blob top-center (`#5E6AD2` at ~8% opacity, heavy blur) + one secondary cooler blob. A faint dot/line grid (`radial-gradient` repeat) at ~3% opacity, masked to fade out below the hero. No `landing-bg.png`, no `Image` import.

2. **Hero** (centered, `pt-40 pb-24`, `max-w-3xl`)
   - Tiny pill badge above headline: `● Live registry` (Linear-style status pill, not the old gradient badge).
   - Headline: one weight (font-semibold, not `extrabold`), `text-5xl/6xl`, tight tracking. **No gradient text** — solid `#F7F8F8`. (Stripe/Vercel both use solid-color headlines; gradient text reads as "AI default".)
   - Subhead in `--muted` (`#8A8F98`), `text-lg`, `max-w-xl`.
   - Two CTAs: primary solid `#5E6AD2` with hover lighten + subtle glow; secondary ghost (`border-white/10`, transparent bg). Replace `rounded-full` → `rounded-lg`, remove `-translate-y-0.5` hover lift (too playful; Stripe uses color/brightness shift only).

3. **Logo strip / trust row** (Linear-style) — small muted "Trusted by" line + a row of muted placeholder wordmarks (Thai Inter Flying + 3 departments as text). Optional but cheap signal of polish. Keep it if it reads well, drop if forced.

4. **Features** — keep the existing 4-card content (QR scan, real-time status, approvals, maintenance) but restyle:
   - Drop `rounded-3xl` → `rounded-xl`, drop per-card colored glows (amber/emerald/indigo icon tiles).
   - Single neutral icon treatment: `w-10 h-10` tile, `bg-white/[0.03]`, `border-white/10`, icon in `#F7F8F8` or muted — accent color appears only on hover.
   - Card hover: border `white/15`, very slight bg lift, no `translate-y`. 200ms `cubic-bezier(0.16,1,0.3,1)`.
   - Keep the bento 2+1+1+2 layout — it's good and on-trend (Vercel/Linear both use asymmetric bento).

5. **Manuals & Support** — two-column kept, restyle to match: hairline-bordered list rows instead of rounded-2xl tiles, monospace codes for "PDF • 4.2 MB" metadata, support cards as flat hairline panels. Strip the per-channel color coding (sky/indigo headers) → all neutral, accent only on hover/icon.

6. **Footer** — flatten: thinner top border (`white/8`), smaller text, muted links. Keep the © 2026 + login + company website links.
